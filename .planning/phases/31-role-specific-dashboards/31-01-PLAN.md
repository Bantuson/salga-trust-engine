Return {"status": scorecard.status, "id": str(scorecard_id)}.

    **Councillor Dashboard (`get_councillor_dashboard`):**
    - Read-only: SDBIPKpi list with traffic light from latest actuals + StatutoryReport list.

    **Audit Committee Dashboard (`get_audit_committee_dashboard`):**
    - All StatutoryReport rows for the tenant.
    - AuditLog entries WHERE table_name IN ('sdbip_scorecards', 'sdbip_kpis', 'sdbip_actuals', 'statutory_reports', 'performance_agreements', 'evidence_documents'), ordered by created_at DESC, limit 100.

    - Query EvidenceDocument WHERE verification_status = 'unverified', joined with SDBIPActual and SDBIPKpi to get KPI context. Return grouped by KPI: {kpi_id, kpi_description, evidence_items: [{id, file_name, content_type, uploaded_at, actual_id}]}.

    **verify_evidence:**
    - Load EvidenceDocument by ID. Update verification_status to 'verified' or 'insufficient'. Create AuditLog with table_name='evidence_documents', operation=UPDATE, changes=json.dumps({"action": "poe_verified" or "poe_insufficient", "evidence_id": str(evidence_id)}).

    **MPAC Dashboard (`get_mpac_dashboard`):**
    - StatutoryReport list for tenant.
    - Investigation flags: Query AuditLog WHERE table_name='investigation_flags', ordered by created_at DESC. For each, JSON-parse `changes` to extract reason, notes, status. The current status of a flag = latest AuditLog row for that record_id (append-only pattern).

    **flag_investigation:**
    - Create AuditLog with table_name='investigation_flags', operation=CREATE, record_id=report_id, changes=json.dumps({"action": "investigation_flagged", "reason": reason, "notes": notes, "report_id": str(report_id), "status": "pending"}).

    - Join with Municipality (NonTenantModel) for name, category, province.
    - Return ranked list by overall_achievement DESC.

    - Query SDBIPKpi WHERE department_id = dept.id joined with latest SDBIPActual.
    - Return: department_name, kpi_count, green/amber/red counts, total_achievement_pct, kpi_details list.
