## Technical Implementation

### LoginPage Enhancements
1. **Title Header**: Added loginHeader and loginTitle styles with proper spacing and typography
2. **Link Button Color**: Changed linkButton style from teal to gold for better visibility
3. **Check Mark Icons**: Removed circle background (borderRadius, backgroundColor) and increased font size to 1.125rem for better visibility
4. **Card Gap**: Increased cardsRow gap by 50% for better visual balance

### RequestAccessPage Enhancements
1. **Header Visibility**: Updated title gradient to gold-to-teal (higher contrast on pink background)
2. **Form Label Styling**: Added CSS class override (.request-access-form) to change all labels to gold without modifying shared Input component
3. **Province Selector Glassmorphism**:
   - Background: rgba(255, 255, 255, 0.15) with 10px blur
   - Custom SVG dropdown arrow in gold (#ffd54f)
   - Removed default select appearance
   - Added className="province-select" for CSS targeting
4. **Submit Button CTA Styling**:
   - Replaced Button component with native button element
   - Gold background with dark text (#333)
   - Glow shadow effect (0 0 20px and 0 0 60px rgba layers)
   - Hover state: transform translateY(-2px) + enhanced glow
   - Added className="request-access-submit" for CSS targeting
5. **Province Options**: Added CSS for option elements with pink background for consistency

### CSS Additions
