## What Was Built

### ResolutionTrendChart (LineChart)
- Recharts LineChart with `MonthlyTrend[]` data
- XAxis tickFormatter strips year, shows month number only (e.g. "02")
- YAxis domain [0, 100] with % formatter
- Glassmorphism tooltip with `rgba(163, 72, 102, 0.95)` background
- Teal stroke (`var(--color-teal)`) with dot r=3, activeDot r=5
- Empty state for zero-length trend arrays

### CategoryDonutChart (PieChart donut)
- Recharts PieChart with innerRadius=60, outerRadius=100, paddingAngle=2
- 6-color teal-first palette: `['#00bfa5', '#ffd54f', '#ff6e40', '#7c4dff', '#ef4444', '#10b981']`
- Custom flex-wrap legend div below chart (avoids Recharts built-in Legend overflow)
- Each legend item: colored dot + "Category (count)"
- Glassmorphism tooltip

### SdbipGaugeChart (RadialBarChart)
- Semi-circle gauge: `startAngle={180} endAngle={0}`, `cy="80%"` to shift arc upward
- Traffic light color: green >= 80%, amber >= 50%, red < 50%
- Background track: `rgba(255,255,255,0.1)`
- Centered percentage + municipality name below arc
- Height=180, innerRadius=60, outerRadius=90, barSize=14

### MunicipalityLeaderboard (sortable table)
- Merges responseTimes + resolutionRates + sdbipData by municipality_id
- Default sort: `resolution_rate` (descending)
- Columns: Rank, Municipality, Resolution Rate, Response Time, SDBIP Achievement, Total Tickets
- Click any column header to re-sort (active column highlighted in teal with underline)
- `avg_response_hours` sorts ascending (lower = better), all others descending
- Position delta: ▲ N (green) / ▼ N (red) / — (unchanged or unknown)
- Row highlight: top 3 green (`rgba(16,185,129,0.12)` + left border), bottom 3 red when total > 5
- Falls back to mockLeaderboard when no real data available

### ServicePerformanceStats (refactored)
- All-municipalities branch: returns `null` — leaderboard owns that view
- Single municipality: stat cards (response time + resolution rate) unchanged
- Replaced horizontal bar chart with `<CategoryDonutChart>`
- Added `<ResolutionTrendChart>` section with glassmorphism card wrapper
- Color text uses `var(--text-primary)` / `var(--text-secondary)` CSS variables

### TransparencyDashboardPage (orchestration)
- Added `financialYear` and `quarter` state (defaulted from `getSAFinancialYear` / `getCurrentSAQuarter`)
- `useResponseTimes`, `useResolutionRates`, `useCategoryBreakdown` all receive `financialYear, quarter`
- `useSdbipAchievement` receives NO quarter/year (annual data — locked decision)
- `<TimePeriodControls>` rendered 16px below MunicipalitySelector
- Conditional main section: `<MunicipalityLeaderboard>` when no municipality, `<ServicePerformanceStats>` otherwise
- SDBIP section replaced linear progress bars with `<SdbipGaugeChart>` per municipality
- Added italic note: "Annual performance data — not affected by quarter filter"
- Privacy Notice preserved unchanged at bottom
