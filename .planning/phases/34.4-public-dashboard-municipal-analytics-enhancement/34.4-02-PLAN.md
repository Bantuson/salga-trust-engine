**3. SdbipGaugeChart.tsx** — RadialBarChart semi-circle gauge.
Props: `{ achievementPct: number; municipalityName: string; }`
- Import `RadialBarChart, RadialBar, ResponsiveContainer` from `recharts`
- Traffic light color: `pct >= 80 ? '#10b981' : pct >= 50 ? '#f59e0b' : '#ef4444'`
- Gauge data: `[{ name: 'achievement', value: achievementPct, fill: gaugeColor }]`
- CRITICAL: use `startAngle={180} endAngle={0}` for semi-circle gauge (per RESEARCH.md Pitfall 1)
- Set `cy="80%"` to shift center upward so gauge sits well in the card
- innerRadius={60}, outerRadius={90}, barSize={14}
- RadialBar: dataKey="value", cornerRadius={6}, background={{ fill: 'rgba(255,255,255,0.1)' }}
- Below the chart, render the percentage value centered with the traffic light color, and the municipality name
- ResponsiveContainer height={180}

**4. MunicipalityLeaderboard.tsx** — Sortable ranking table with position deltas.
Props: `{ responseTimes: ResponseTimeData[]; resolutionRates: ResolutionRateData[]; sdbipData: SdbipAchievementData[]; }`
- Import `LeaderboardEntry` type from `../types/public`
- Import `mockLeaderboard` from `../data/mockDashboardData` for fallback
- Internal state: `sortKey` with type `'resolution_rate' | 'avg_response_hours' | 'sdbip_achievement_pct' | 'total_tickets'`, default `'resolution_rate'`
- `computeLeaderboard` function: merge responseTimes + resolutionRates + sdbipData by municipality_id, sort by sortKey (descending for rates/pct/tickets, ascending for response_hours since lower is better), assign current_rank. For previous_rank and rank_delta, look up from mockLeaderboard by municipality_id (since real historical data is not available yet). If no match, set previous_rank=null, rank_delta=null. If real data produces empty result, fallback to mockLeaderboard.
- `useMemo` to recompute leaderboard when data or sortKey changes
- Column headers: Rank, Municipality, Response Time, Resolution Rate, SDBIP Achievement, Total Tickets. Each header is clickable to change sortKey (show subtle underline or bold on active column).
- Row rendering:
  - Rank column: show `current_rank` number
  - Position delta: show arrow icon (triangle up/down character) with delta number. `delta > 0` = green "up N", `delta < 0` = red "down N", `delta === 0` = grey dash, `null` = empty. Use characters: up arrow = "▲", down arrow = "▼", no change = "—"
  - Row highlight: top 3 rows get `background: 'rgba(16, 185, 129, 0.12)'` with `borderLeft: '3px solid #10b981'`. Bottom 3 rows (when total > 5) get `background: 'rgba(239, 68, 68, 0.12)'` with `borderLeft: '3px solid #ef4444'`.
- Use GlassCard wrapper with glassmorphism styling
- All inline styles, CSS variables for text colors
- Table header uses slightly darker background: `rgba(255,255,255,0.05)`
