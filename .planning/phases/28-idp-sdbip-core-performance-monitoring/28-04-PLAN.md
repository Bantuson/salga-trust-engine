From research — compute_achievement pattern:
```python
from decimal import Decimal

def compute_achievement(actual: Decimal, target: Decimal) -> tuple[Decimal, str]:
    if target == 0:
        return Decimal("0"), "red"
    pct = (actual / target) * Decimal("100")
    if pct >= Decimal("80"):
        return pct, "green"
    elif pct >= Decimal("50"):
        return pct, "amber"
    else:
        return pct, "red"
```
