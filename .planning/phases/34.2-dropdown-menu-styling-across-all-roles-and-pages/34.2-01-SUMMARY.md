## Decisions Made
- SIZE_STYLES[sm] uses 28px right padding (not 2.5rem) to accommodate chevron in compact navbar contexts
- Selected item background highlight rgba(0,191,165,0.25) added to match GlassSelect's optionActive style — previously the existing Select only changed color, not background
- RoleSwitcher container stripped of surface-elevated pill styling (background, backdropFilter, border removed) — unified Select sm size carries its own glass border
- TeamDefaultsSection maxWidth='320px' handled via wrapper div — GlassSelect's maxWidth prop not ported to unified Select to keep the API clean
- autoFlip threshold: spaceBelow < 248 (240px maxHeight + 8px buffer as specified in plan)
