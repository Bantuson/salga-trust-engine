## Decisions Made

- Pattern A vs B determination by inspecting which styles object has `overflowY: auto` — if `styles.body` has it, add to body div (Pattern A); if `styles.modal` has it, add to modal div (Pattern B)
- One `data-lenis-prevent` per modal on the outermost scrollable container is sufficient; Lenis traverses upward through the DOM so all descendant scroll targets are covered
- `data-lenis-prevent` written as a bare JSX attribute (no `={true}`) matching the Lenis API convention
