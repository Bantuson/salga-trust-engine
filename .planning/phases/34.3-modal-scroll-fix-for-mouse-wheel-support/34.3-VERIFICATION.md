### 1. Mouse Wheel Scrolling in Fixed Modals

**Test:** Open the TeamCreateModal (navigate to `/teams`, click "Create Team"). Mouse-wheel inside the modal body while the modal is open.
**Expected:** The modal body scrolls down through its form fields. The page background behind the modal does not scroll.
**Why human:** Lenis scroll interception and `data-lenis-prevent` behaviour can only be observed in a live browser with Lenis running. Playwright's `page.mouse.wheel()` in the E2E spec exercises the assertion but requires a running dev server to produce a meaningful pass verdict.

**Test:** Open any fixed modal (e.g. TeamDetailModal, InviteUserModal). Inspect that the glass backdrop blur, border, and visual styling are intact.
**Expected:** Modal appears visually unchanged — glass blur and border render as before. The `data-lenis-prevent` attribute addition does not affect CSS.
**Why human:** Visual regression cannot be verified programmatically. The attribute is a non-CSS HTML data attribute so it should have no visual effect, but confirmation is straightforward.
