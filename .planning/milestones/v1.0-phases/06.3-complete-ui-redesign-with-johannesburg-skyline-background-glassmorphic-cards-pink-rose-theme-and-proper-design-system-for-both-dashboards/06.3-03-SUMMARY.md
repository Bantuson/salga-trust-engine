# Phase 06.3 Plan 03: Johannesburg Skyline Hero Background Summary

Johannesburg skyline photograph replaces 3D globe as landing page hero background with GSAP scroll-driven fade-out to solid pink and pink border frame overlay.

## What Was Built

Replaced the Globe3D component in the public landing page hero section with the Johannesburg skyline photograph from `design-references/background+pink-border.png`. Implemented GSAP ScrollTrigger animation that fades the skyline to opacity 0 as the user scrolls, revealing solid pink (#cd5e81) background behind it. Added a pink border frame as a separate UI element overlay.

## Technical Details

**GSAP Animation Implementation:**
```tsx
gsap.to(skylineRef.current, {
  opacity: 0,
  ease: 'none',
  scrollTrigger: {
    trigger: sectionRef.current,
    start: 'top top',
    end: 'bottom top',
    scrub: true,
  },
});
```

**Layer Stack (bottom to top):**
1. Layer 1: Solid pink background (`.hero-section` background: var(--surface-base))
2. Layer 2: Skyline photograph (`.skyline-background`, z-index: 0)
3. Layer 3: Pink border frame (`.skyline-frame`, z-index: 2)
4. Layer 4: Hero content (`.hero-content`, z-index: 3)
5. Layer 5: Gradient overlay (`.hero-gradient-overlay`, z-index: 3)

**Responsive Adjustments:**
- Mobile (<768px): Skyline positioned at `center 60%` (lower in frame for better text readability)
- Mobile (<768px): Border width reduced from 20px to 12px (prevents excessive border on small screens)
