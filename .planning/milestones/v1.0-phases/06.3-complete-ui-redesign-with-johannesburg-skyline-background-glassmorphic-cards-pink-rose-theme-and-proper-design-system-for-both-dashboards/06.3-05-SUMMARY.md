## Technical Notes

**Skyline Background Architecture:**
Three fixed layers create the full skyline effect:
1. `.auth-skyline-bg` — The skyline image (background-position: center bottom)
2. `.auth-skyline-overlay` — Pink tint overlay (rgba(205, 94, 129, 0.3))
3. `.auth-skyline-frame` — Pink border frame (rgba(205, 94, 129, 0.4))

All layers use `position: fixed; inset: 0; z-index: 0` (except frame at z-index: 1) to stay behind glassmorphic cards.

**Color Token Cascade:**
Municipal dashboard now fully relies on shared design tokens:
- `var(--surface-base)`, `var(--surface-elevated)`, `var(--surface-higher)` auto-cascade to pink variants
- `var(--color-accent-gold)` for primary actions (replaces coral)
- `var(--color-teal)` remains for focus states and status indicators (works well on pink)
- Semantic colors (coral for danger, gold for warning) preserved for UX clarity

**Visual Parity Achieved:**
Both dashboards now share identical:
- Background colors (pink/rose)
- Glassmorphic card styling
- Button color schemes (gold primary, teal secondary)
- Table styling (rose header tint)
- Scrollbar styling
